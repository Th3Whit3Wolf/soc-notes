## System and OS Information
---

###  OS Release Information
<FRLoc location="/etc/os-release">
Can be read using `cat`, `vim` or any text editor or viewer
</FRLoc>

###  User Accounts Information

<FRLoc location="/etc/passwd">
Can be read using `cat`, `vim` or any text editor or viewer.
</FRLoc>

###  User Group Information

<FRLoc location="/etc/group">
Can be read using `cat`, `vim` or any text editor or viewer.
</FRLoc>

###  Sudoers List

<FRLoc location="/etc/sudoers">
Can be read using `cat`, `vim` or any text editor or viewer.
</FRLoc>

###  Login Information

<FRLoc location="/var/log/wtmp">
Can be read using `last` utility
</FRLoc>

###  Authentication Logs

<FRLoc location="/var/log/auth.log">
Can be read using `cat`, `vim` or any text editor or viewer.
Use `grep` for better filtering.
Might also have `auth.log1`, `auth.log2` etc as log files that have been rotated.
</FRLoc>

## Evidence of Execution
---

###  Authentication Logs

<FRLoc location="/var/log/auth.log">
```bash
/var/log/auth.log* |grep -i COMMAND;
```
the `grep` can be used to filter the results. Can be read using `cat`,
`vim` or any text editor or viewer
</FRLoc>

## Hashes

### Generate MD5 Hash of a File

```bash
md5sum <file>
```

### Generate SHA1 Hash of a File

```bash
sha1sum <file>
```

### Generate SHA256 Hash of a File

```bash
sha256sum <file>
```
